Couch Meets Table

Easy-to-use roomate matching platform for CWRU students.

Django MySQL JavaScript Python HTML CSS
Role: Frontend Lead, Developer
Team Size: 4 people
Release: 2025

Couch Meets Table is a full-stack web application designed to help Case Western Reserve students find compatible roommates. The platform allows users to create profiles, specify living habits and roommate preferences (including dealbreakers), and swipe through potential matches ranked by our compatibility algorithm.

Responsibilities

Frontend Lead, Developer

  • Led frontend development using React and Tailwind CSS.
  • Designed application UI/UX in Figma to ensure cohesive user experience.
  • Implemented login, signup, create profile, roommate quiz, and matches pages.
  • Integrated frontend components with backend APIs.
