To-Do List

React-based task management application for users to add, update, and delete tasks.

React JavaScript CSS TypeScript HTML
Check it out

Responsibilities

  • Used React Hooks for state management and reusable components.
  • Integrated Material UI components for a responsive UI and modal-based forms.
  • Added form validation, priority selection, and deadline tracking for user experience.
  • Implemented toastr notifications for success feedback on add, update, and delete actions.

Process

For this project, I began by outlining the core features and user interactions. This included task creation, editing, deletion, priority levels, and deadlines. Based on these requirements, I designed reusable components for task items, modals, and form inputs. I also structured state management using React Hooks to handle dynamic updates to the task list and form states. I also integrated Material UI components to for styling and responsive design. I also implemented toast notifications and form validation. Finally, I tested the application, including edge cases such as empty inputs, to ensure a clear and user-friendly experience.
