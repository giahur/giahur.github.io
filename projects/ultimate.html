Ultimate

Website to play Ultimate Tic-Tac-Toe with a friend!

React JavaScript CSS TypeScript HTML
Play

Ultimate Tic-Tac-Toe is a complex version of the classic game with nine 3x3 boards to create an ultimate large board. Each of your moves will determine where your opponent can play next. Wins on the smaller boards count as moves on the larger board. Win the whole game by getting 3-in-a-row on the large board!

Responsibilities

  • Designed and implemented a component-based React architecture.
  • Built complex game logic, including multi-board state management, move restrictions, and conditional board activation.
  • Managed shared and local state using React Hooks to coordinate turns, board activity, and game outcomes.
  • Implemented client-side routing with React Router for easy navigation.
  • Styled the application with CSS, focusing on visual feedback for a complete gameplay experience.
