Ultimate

Play

Website to play Ultimate Tic-Tac-Toe with a friend! Made using React, JavaScript, CSS, TypeScript, and HTML

Ultimate Tic-Tac-Toe is a complex version of the classic game with nine 3x3 boards to create an ultimate large board. Each of your moves will determine where your opponent can play next. Wins on the smaller boards count as moves on the larger board. Win the whole game by getting 3-in-a-row on the large board!

I developed this interactive web application using React framework. I also incorporated React Hooks (state management) to handle game logic such as tracking and updating game state across multiple boards, handling tie conditions, and indicating the next allowed moves.
